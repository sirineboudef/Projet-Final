importer_vent module¶

Ce module définit la classe ImportVent.

Responsable de :

  • récupérer les données de vent en temps réel via l’API Open-Meteo,

  • interpoler les composantes du vent en fonction de l’altitude et du temps,

  • renvoyer le champ de vent utilisé dans l’optimisation de trajectoire.

class importer_vent.ImportVent(lat, lon, hour_index=0, N=31, z0=1200)¶

Bases: object

Classe qui permet l’interpolation des vents à différentes altitudes à partir de l’API Open-Meteo.

Parameters:

  • lat (float) – Latitude de la zone cible.

  • lon (float) – Longitude de la zone cible.

  • hour_index (int) – Index temporel dans les données de vent (ex: 0 = heure actuelle).

  • N (int) – Nombre d’étapes de discrétisation temporelle.

  • z0 (float) – Altitude initiale (ex: altitude d’ouverture du parachute).

Variables:

  • vx_interp – Composante horizontale du vent interpolée.

  • vy_interp – Composante verticale du vent interpolée.

import_vent()¶

Récupère et interpole les données de vent fournies par l’API Open-Meteo.
